# Cluster View

[広聴AI](https://github.com/digitaldemocracy/kouchou-ai)が出力する hierarchical_result.json をアップロードして可視化・共有するWebアプリケーションです。

## 必要条件

- [Deno](https://docs.deno.com/runtime/getting_started/installation) 2.0以上

## 開発

```bash
deno task dev
```

## 本番ビルド

```bash
deno task build
deno task start
```
